# Getting Started with Astro: A Modern Web Framework

Astro is a revolutionary web framework that's changing how we think about building websites. With its unique approach to partial hydration and component islands, Astro delivers exceptional performance while maintaining developer experience.

## Key Features

- **Zero JS by default**: Only ship JavaScript when you need it
- **Framework agnostic**: Use React, Vue, Svelte, or plain HTML
- **Built-in optimizations**: Automatic image optimization, CSS bundling
- **Developer experience**: Hot reloading, TypeScript support, and more

## Getting Started

```bash
npm create astro@latest my-project
cd my-project
npm run dev
```
